# 🎥 Vidsnap AI

Vidsnap AI is a Python Flask-based web application that enables users to download and manage online video content through a clean and user-friendly interface. The project demonstrates backend development using Flask while providing a responsive frontend for an improved user experience.

---

## 🚀 Features

- Download videos from supported platforms
- Simple and responsive user interface
- Fast video processing
- Clean Flask backend
- Easy to use and lightweight
- Organized project structure

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Vidsnap-AI.git
```

### Navigate to the project

```bash
cd Vidsnap-AI
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

### Activate the virtual environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
